Provides a tree-sitter grammar for InterSystems ObjectScript, enabling fast and precise parsing for use in code editors, linters, and developer tooling.

The grammar currently integrates with these editors: 
- [Zed](https://zed.dev/extensions/objectscript)
- [Neovim](https://neovim.io)
- [Emacs](https://github.com/intersystems/emacs-objectscript-ts-mode)
